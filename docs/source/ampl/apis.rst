Python
------

- Installation:

.. code-block:: bash

    $ python -m pip install amplpy

- Documentation: https://amplpy.ampl.com/
- GitHub: https://github.com/ampl/amplpy

R
-

- Installation:

- GitHub: https://github.com/ampl/rAMPL
